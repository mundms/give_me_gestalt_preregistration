Step 5.Test assumptions for linear regression.

Normality:

# Non-averaged data:
ks.test(b1_exp$response, pnorm)
## 
##  One-sample Kolmogorov-Smirnov test
## 
## data:  b1_exp$response
## D = 0.91614, p-value < 2.2e-16
## alternative hypothesis: two-sided
ks.test(b2_exp$response, pnorm)
## 
##  One-sample Kolmogorov-Smirnov test
## 
## data:  b2_exp$response
## D = 0.86799, p-value < 2.2e-16
## alternative hypothesis: two-sided
# Averaged data:
ks.test(b1_exp_mean$mean_Like_exp, pnorm)
## 
##  One-sample Kolmogorov-Smirnov test
## 
## data:  b1_exp_mean$mean_Like_exp
## D = 0.99275, p-value < 2.2e-16
## alternative hypothesis: two-sided
ks.test(b2_exp_mean$mean_Detect_exp, pnorm)
## 
##  One-sample Kolmogorov-Smirnov test
## 
## data:  b2_exp_mean$mean_Detect_exp
## D = 0.98687, p-value < 2.2e-16
## alternative hypothesis: two-sided

Result: According to the Kolmogorov-Smirnov test the variables are not normal distributed. Hence, the normality assumption is not fullfiled. We will test the residual distribution in the next step to make sure that we can still use a linear regression.
